# LINQ to SQL (Sample)
A sample project to explain how .NET LINQ Queryable providers work.

## Início rápido
- Install [.NET SDK](https://dotnet.microsoft.com) version 6 or later (see `global.json`).
- Navigate to the sample project: `cd src/Sample`.
- Run `dotnet run`.
